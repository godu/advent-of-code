# advent-of-code-2020

[Advent of code](https://adventofcode.com/2020)



## Install

```shell
$ stack install hlint ormolu
```

## Test

```shell
$ hlint .
$ ormolu --color always --check-idempotence --mode check **/*.hs
$ stack test [--file-watch --fast]
```

## Run

```shell
$ stack run
```
